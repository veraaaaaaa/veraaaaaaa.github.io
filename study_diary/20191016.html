在 vue 專案裡面使用 sass + webpack ( 2019 / 10 / 16 )


延用我建立的 vue + webpack + stylus 專案，把載入的 stylus 改成載入 sass 的東西

npm install node-sass --save-dev
npm install sass-loader --save-dev

然後 webpack.config.js 裡面的 stylus 設置改成下面的樣子

{
    test: /\.scss$/,
    loaders: ['style-loader', 'css-loader', 'sass-loader']
}

我這邊踩到的坑是我本來寫的是沒有 loader 的版本，他的錯誤提醒會跟你說需要加上去

{
    test: /\.scss$/,
    loaders: ['style', 'css', 'sass']
}

在 vue2.x 裡面用

 <style lang="scss" scoped="" type="text/css">

新增 style.sass 的話需要在 index.js 裡面 import 位置

 import './assets/styles/style.scss'

我朋友踩到的坑是 sass-loader 的版本太高

1.將package.json中的sass-loader的配置修改成低於8.0.0版本，修改成7.3.1
2.將安裝的node_modules項目依賴刪除，重新安装，將sass-loader版本變成7.3.1
3.重新運行項目 npm run dev





建議使用 Chrome、Firefox、Safari 瀏覽
